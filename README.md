# 遠隔患者モニタリングシステム

## これは何？

自宅療養や宿泊療養中の患者にスマートフォン上で必要項目を入力してもらい、患者の健康状態の把握を保健所で行いやすくするためのシステムです。

背景情報についてぇあ、[BACKGROUND.md](docs/BACKGROUND.md)もご覧ください。
